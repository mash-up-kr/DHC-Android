+ Architecture : MVI (no library)
+ UI : Compose
+ Di : Dagger Hilt
+ Network : retrofit
+ Image : coil
+ Video : media3 exoplayer
+ 직렬화/역직렬화 : Kotlin Serialization
+ 저장소: Proto DataStore

# 🛠 Tech & Library

+ Target SDK Level 35
+ Min SDK Level 26
+ Kotlin 2.0.21

| 구분                      | 내용                                                         |
|-------------------------|------------------------------------------------------------|
| Architecture            | Google Recommend Architecture                              |
| DesignPattern           | MVI                                                        |
| Jetpack                 | Compose, Lifecycle, Navigation, Proto DataStore, ViewModel |
| Network                 | Retrofit                                                   |
| Asynchronous Processing | Coroutine, Flow                                            |
| Dependency Injection    | Hilt                                                       |
| Third Party Library     | Coil-Compose, Firebase Analytics                           |
| Other Tool              | Figma                                                      |
